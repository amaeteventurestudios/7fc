# 7FC Image Asset Manifest

All assets are **WebP only** (no PNG/JPEG/JPG). State `final` means real,
approved artwork is wired into the site. State `placeholder` means the file
still contains the auto-generated placeholder graphic (dark navy/black card
with the filename, purpose, and dimensions printed on it) and must be
replaced before launch. State `deprecated` means the file exists but is no
longer rendered anywhere on the site.

## Regenerating placeholders

```bash
node scripts/generate-placeholders.mjs
```

## Replacing with final art

1. Export final artwork as WebP at the listed dimensions (or larger, same aspect ratio).
2. Overwrite the file in `/public/images/` keeping the exact filename.
3. No code changes required.

Country flags are rendered as emoji in V1 — no flag image files are used.
